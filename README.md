# 🏨 Hotel Management System

A modern and responsive **Hotel Management System Website** developed using **HTML, CSS, and JavaScript**. This project simulates a real-world hotel booking platform with an attractive user interface and interactive functionality.

---

## 🚀 Features

- 🏨 Modern Home Page
- 🛏️ Room Listings
- 📅 Online Room Booking
- 👤 Customer Registration
- 📋 Booking Management
- 💳 Booking Summary
- 📱 Fully Responsive Design
- 🌙 Dark Mode
- 🔍 Room Search & Filters
- 📞 Contact Form
- ⭐ Modern UI/UX

---

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript
